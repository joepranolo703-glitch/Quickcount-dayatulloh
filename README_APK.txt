
# Quick Count Android APK

## Cara Build APK Android

### 1. Install Node.js
Download:
https://nodejs.org

### 2. Install Android Studio
Download:
https://developer.android.com/studio

### 3. Install dependency
Buka terminal di folder project:

npm install

### 4. Build project
npm run build

### 5. Tambahkan Android
npx cap add android

### 6. Sync project
npm run sync

### 7. Buka Android Studio
npm run android

### 8. Generate APK
Di Android Studio:
Build > Build APK(s)

APK akan berada di:
android/app/build/outputs/apk/debug/

Nama aplikasi:
Quick Count Dayatulloh
Package:
com.baday.quickcount
